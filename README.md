## Ficheros
Creación de ficheros .rtf, .xml y .html

El primer paso para la creación de 3 ficheros con diferentes extensiones ha sido abrir un editor de texto, escribir un código (en este caso un ejemplo de los apuntes del módulo de *Lenguaje de Marcas y Sistemas de Gestión de la Información*) y guardar dicho fichero como documento de texto.
